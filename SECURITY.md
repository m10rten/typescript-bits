# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in typescript-bits, please report it privately by opening a security advisory on GitHub:

**https://github.com/m10rten/typescript-bits/security/advisories/new**

Do not report security vulnerabilities through public GitHub issues.

We will acknowledge receipt within 72 hours and provide an expected timeline for a fix. You will receive updates as the investigation progresses.

## Scope

This policy applies to the `typescript-bits` npm package, the coresponding website and its source code. Third-party dependencies should be reported to their respective maintainers.

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.x     | ✅ Yes    |
